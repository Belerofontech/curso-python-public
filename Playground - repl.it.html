# Repl.it playgrounds

      : https://www.repl.it
      .
      Hay para muchos lenguajes! (elegir arriba "Start coding")
      No efímeros (si guardamos la URL que nos cree -con un 
      nombre "inventado"- y/o creamos una cuenta gratuita)
      .
      Además permiten que se trabaje de forma colaborativa!
      Ojo: caducan (?), son *públicos* (salvo con cuenta de pago),
      y lógicamente no debemos hacer o guardar nada sensible!
      O sea, muy similar en filosofía a GitHub...
      Doc. oficial:
      : https://docs.repl.it/misc/quick-start

      Más ventajas: cómodo, tiene un IDE "Monaco" (editor, debugger),
      tiene más herramientas tipo Git y MongoDB (sólo con una cuenta,
      gratuita o no), incluyendo leer/usar repos GitHub, también tiene
      gestión de paquetes...
      Y hay una comunidad: ejemplos, tutoriales, "concursos" (?)...
      .
      Ejemplos curiosos e interesantes:
      : https://repl.it/talk/share/A-sudoku-solver-I-made/56706
      : https://repl.it/talk/share/Language-Whatifyer/50169

      Y la siguiente slide/página "integra" ya un repl concreto "mío",
      que se podría abrir simplemente en una nueva pestaña también:
      : https://repl.it/@pachibelero/LimpingLinearMotion?lite=true
